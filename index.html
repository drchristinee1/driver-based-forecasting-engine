Driver-Based Forecasting Engine

A FinOps forecasting prototype that translates workload demand into cloud cost and unit economics.

Architecture

Business Demand → Infrastructure Usage → Pricing → Forecasted Cost → Unit Economics

Example Input

{ "transactions": 1000000, "api_requests": 5000000, "storage_gb": 20000, "growth_rate": 0.15 }

Example Output

{ "projected_total_cost": 10560.4, "cost_per_transaction": 0.009183 }

Run Locally

python main.py
